Start with the real use case, not the UTM jargon. Examples: `Newsletter` for list sends, `In-person / QR swag` for card protectors or table tents, `Card room / casino floor` for live-room giveaways.

Think “which platform or system sent the click?” Examples: `google`, `youtube`, `newsletter`, `apollo`, `offline`, `cardroom`, `live-event`.

Think “what type of traffic was it?” Examples: `cpc`, `paid_social`, `email`, `cold_email`, `organic_video`, `qr`, `offline`, `in_person`.

Use the thing the user actually clicked. Examples: `email`, `video`, `creator`, `link`, `swag` (merch with QR), `print`, `booth`.

Think “where exactly was this link?” Good examples: `main-cta`, `pinned-comment`, `description-main-cta`, `creator-description`, `post-link`, `footer-button`.

Most of the time this just stays `v1`. Move to `v2` or `v3` only if you changed the creative, copy, link location, or CTA enough that you want it split in reporting.

Skip this for most links. Good uses: Google Ads keyword, subreddit name, email segment, or paid audience. If you are unsure, leave it blank.

Auto-suggested. Required for paid, partner, affiliate, and cold outbound links.

How Channel, Source, And Medium Work Together
Channel: the practical marketing use case you are working on.
Source: the platform or system that sent the click.
Medium: the traffic type.
Google Ads: channel = `Google Ads`, source = `google`, medium = `cpc`
Newsletter send: channel = `Newsletter`, source = `newsletter`, medium = `email`
Owned YouTube link: channel = `Owned YouTube`, source = `youtube`, medium = `organic_video`
Partner YouTube placement: channel = `Partner YouTube`, source = `youtube`, medium = `partner`
Cold outbound email: channel = `Cold Email`, source = `apollo`, medium = `cold_email`
QR on swag: channel = `In-person / QR swag`, source = `offline`, medium = `qr`
Card room giveaway: channel = `Card room / casino floor`, source = `cardroom`, medium = `in_person`
How To Think About These Fields
Link type / channel: What kind of marketing activity is this?
UTM source: Which platform or system sent the click?
UTM medium: What type of traffic is it?
Asset type: What kind of thing did they click?
Placement / hook: Where exactly did that link appear?
Variant: Which version was it?
UTM term: Extra targeting detail only if it will matter later.
UTM ID: Unique tracking ID. Usually just accept the tool's suggestion.
